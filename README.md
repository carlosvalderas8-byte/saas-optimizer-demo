# SaaS Optimizer - Demo

**Optimiza y gestiona tus suscripciones SaaS** de forma inteligente.

## Características

- 📊 **Dashboard** - Vista general de todas tus suscripciones
- 📋 **Gestión de Suscripciones** - CRUD completo de servicios
- 🔍 **AI Discovery** - Detecta duplicidades y oportunidades de ahorro
- ⚡ **AI Optimizer** - Recomendaciones inteligentes para optimizar gastos
- 📅 **Calendario de Renovaciones** - No pierdas de vista los próximos pagos
- 📈 **Analytics** - Visualiza y analiza tus gastos
- ✅ **Compliance** - Verifica el cumplimiento de políticas

## Requisitos

- [Node.js](https://nodejs.org/) 18+ (versión LTS recomendada)

## Instalación Rápida

1. Descarga y extrae el ZIP
2. Haz doble clic en `INICIAR_APP.bat`
3. Abre http://localhost:3000 en tu navegador
4. Regístrate con tu email (sin contraseña)
